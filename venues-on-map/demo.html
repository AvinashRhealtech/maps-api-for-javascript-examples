Display Venues

Using the Venues module facilitates access to given indoor maps including full JSON models. The Venues APIs provide information about the indoor spaces, including buildings, level geometry and points of interest.

More information about the use of Venues API can be found here.

This example shows a map of Zurich Airport.

Code

The example shows loading of indoor information using H.venues.Service and renders the indoor map using an instance of H.venues.Provider.
It also shows usage of venue related ui controls to change levels and buildings using respectively H.venues.ui.LevelControl and H.venues.ui.DrawingControl.
